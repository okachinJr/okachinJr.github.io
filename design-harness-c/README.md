# デザインハーネス — GBレトロホームページ型（案Cスタイル）

初代ゲームボーイ×90年代個人ホームページ風の1ページサイトを、デザインを崩さずに作成するための持ち出し用キット。
**新しいサイトを作るときは、このフォルダごとコピーして `template.html` を出発点にすること。ゼロから書き起こさない。**

## キットの構成

| ファイル | 役割 |
|---|---|
| `README.md` | この仕様書。実装エージェントへの規則 |
| `template.html` | 完動するテンプレート（CSS/JS完全インライン1ファイル） |
| `hero-title-screen.png` | サンプルのメインビジュアル。**新サイトでは必ず差し替える** |

## 使い方（エージェント向け手順）

1. `template.html` を新プロジェクトへコピーし `index.html` にリネーム
2. メインビジュアル画像を差し替える（`hero-title-screen.png` 参照箇所。横長4:3推奨、ピクセルアート推奨）
3. `<!-- TODO: URL差し替え -->` の付いた `href="#"` を実URLに差し替える
4. ブランド文字列を一括置換（「おかちんJr.」「OKACHIN JR.」「SINCE 1995」等）
5. リンクカードの枚数・文言を調整（グリッドは自動で折り返す）
6. 末尾の「検証チェックリスト」を全項目実施

## デザイン規則（不変条件）

### 1. カラーパレット — 5色のみ
```css
--gb0: #1e3a20;  /* 最暗・インク・枠線・ティッカー背景 */
--gb1: #4a6b3d;  /* 暗めの中間色・補助テキスト */
--gb2: #8fae5e;  /* 中間色 */
--gb3: #cfe0a0;  /* 明るい中間色・背景ドット柄・バナー背景 */
--gb4: #e4eec6;  /* 最明・ページ背景・ウィンドウ背景 */
```
- この5色以外の色コードを追加しない。半透明も追加しない（スキャンラインは `opacity` で調整）
- メインビジュアルを差し替えた場合は、**画像から5色を取り直してCSS変数を更新してよい**（明→暗の5段階、緑系でなくても可）。ただし色数は5のまま

### 2. タイポグラフィ — 2書体のみ
- 日本語・本文: `'DotGothic16', sans-serif`
- 英字見出し・装飾: `'Press Start 2P', cursive`（見出し・ティッカー・バナー・コピーライト）
- Google Fonts の `<link>` で読み込む。他フォント追加禁止

### 3. 部品カタログ（template.html に実装済み。新セクションもこの部品で組む）
- **RPGウィンドウ** `.rpg-window` — --gb4 背景 + 3px実線 + inset box-shadow の二重枠。四隅に ✦（`.corner-tl` 等の4つの span。**擬似要素で足さないこと** — 位置指定漏れの原因になる）
- **ティッカー** `.ticker-wrap` — --gb0 帯 + Press Start 2P が横に流れる。`prefers-reduced-motion: reduce` では静止表示
- **リンクカード** `.link-card` — 二重枠 + アイコン + 英字名 + 日本語説明。ホバー/フォーカスで**配色反転（トランジションなし、カクッと切替）**+ ▶ カーソル出現
- **88×31バナー** `.banner-88x31` — 昔の相互リンクバナー。CSSのみ（実線枠 + 破線内枠 + 極小 Press Start 2P）
- **スキャンライン** `.scanlines` — `position: fixed` + `repeating-linear-gradient` + `opacity: 0.04` + `pointer-events: none`
- **背景ドット柄** — body の `radial-gradient` 2枚重ね（8px間隔）

### 4. ピクセル表現・モーションの規律
- `border-radius` 禁止、`box-shadow` のぼかし禁止、グラデーション禁止（ドット柄・スキャンライン用の repeating 系のみ可）
- 画像には `image-rendering: pixelated`
- ホバーは即時切替が原則。smooth なトランジション・フェードを足さない
- アニメーションは必ず `@media (prefers-reduced-motion: reduce)` で停止

### 5. ページ構成（セクション順序。増やすのは可、順序変更・削除は要確認）
1. ヘッダー（ロゴテキスト + 英字サブタイトル）
2. ティッカー
3. メインビジュアル（RPGウィンドウ額装）
4. `▼ SELECT ▼` リンクカードグリッド（3列 → 880px以下2列 → 560px以下1列）
5. あいさつウィンドウ
6. フッター（バナー列 + コピーライト）

### 6. 実装の規律
- 1ファイル完結（CSS/JSインライン）。外部ライブラリ・ビルドツール禁止。JSは原則不要
- 外部リンクは `target="_blank" rel="noopener noreferrer"`
- `lang="ja"`、画像に具体的な `alt`、`:focus-visible` アウトラインを消さない
- `console.log` を残さない。ファイル末尾は改行

## 検証チェックリスト（作成・変更のたびに実施）

- [ ] 375px幅で横スクロールが発生しない（`scrollWidth === clientWidth`）
- [ ] コンソールにエラー・警告・ログがない
- [ ] 使用色が定義済み5色のみ
- [ ] リンクカードのホバー/フォーカスで反転 + ▶ が出る
- [ ] `prefers-reduced-motion: reduce` でティッカーが静止する
- [ ] `href="#"` が残っていない（公開前）
